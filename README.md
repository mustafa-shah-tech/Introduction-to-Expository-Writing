# 📝 Introduction to Expository Writing
### Interactive Self-Learning Study Guide

[![Live Site](https://img.shields.io/badge/Live%20Site-Visit%20Now-6366f1?style=for-the-badge&logo=github)](https://mustafa-shah-tech.github.io/Introduction-to-Expository-Writing/)

A beautifully designed, single-file interactive study course for **Introduction to Expository Writing (ENG COMP 101)**. Built with pure HTML, CSS, and vanilla JavaScript — no frameworks, no dependencies, no build step.

---

## ✨ Features

- 🔴 **Exam Alert Boxes** — Animated pulsing alerts highlight high-probability exam questions throughout every chapter
- ✅ **Interactive Mini-Quizzes** — Instant feedback on every answer, with a score summary and retry option after each quiz
- 📊 **Comparison Grids, Timelines & Formula Boxes** — Visual learning components that make abstract concepts easy to scan and remember
- 💾 **Persistent Progress** — Topic completion and quiz scores are saved automatically via `localStorage` and restored on every visit
- 📱 **Fully Mobile-Responsive** — Hamburger menu with animated sidebar overlay, fluid layouts, and optimized typography for all screen sizes
- 🌙 **Academic Night Theme** — Dark-mode-first design with a curated indigo/sky/emerald color palette

---

## 📚 Course Content

| Chapter | Topics Covered |
|---------|---------------|
| **1 — The Paragraph** | Parts of a Paragraph · Principles of Structure (Unity, Coherence, Adequacy) |
| **2 — Types of Writing** | Expository & Narrative · Persuasive & Descriptive |
| **3 — The Essay** | Introduction, Body, Conclusion · The Thesis Statement |
